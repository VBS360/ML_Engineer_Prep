# 🤖 Advanced GenAI Interview Questions (AI Engineer Level)

This document contains high-impact Generative AI questions with **interview-ready answers**.

---

## ❓ 1. What is a Transformer architecture?

A Transformer is a deep learning architecture used in LLMs. It uses a mechanism called attention to understand relationships between words in a sentence.

Unlike older models, it processes all words in parallel, which makes it faster and more efficient.

---

## ❓ 2. What is Attention Mechanism?

Attention helps the model focus on important words in a sentence while processing it.

For example, in a sentence, some words are more important for meaning, and attention assigns higher importance to those words.

---

## ❓ 3. What is Self-Attention?

Self-attention means the model looks at all words in a sentence and understands how each word is related to every other word.

This helps in capturing context better.

---

## ❓ 4. What is Tokenization?

Tokenization is the process of breaking text into smaller units like words or subwords.

These tokens are then used as input to the model.

---

## ❓ 5. What are Tokens?

Tokens are the basic units of text that the model processes.

For example, a sentence can be broken into multiple tokens before being fed into the model.

---

## ❓ 6. What is Context Window?

Context window is the maximum amount of text (tokens) that a model can process at one time.

If the input exceeds this limit, the model may ignore older information.

---

## ❓ 7. What is Prompt Injection?

Prompt injection is a security issue where a user manipulates the prompt to get unintended or harmful outputs from the model.

---

## ❓ 8. What is Zero-shot, One-shot, and Few-shot Learning?

* Zero-shot: No examples provided
* One-shot: One example
* Few-shot: Few examples

These help guide model behavior.

---

## ❓ 9. What is Temperature vs Top-p?

Temperature controls randomness, while top-p controls probability distribution of word selection.

Both are used to control output creativity.

---

## ❓ 10. What is Fine-Tuning vs RAG?

Fine-tuning modifies the model weights, while RAG retrieves external data and combines it with the model’s knowledge.

---

## ❓ 11. What is Latency in GenAI systems?

Latency is the time taken by the model to generate a response.

Lower latency improves user experience.

---

## ❓ 12. What is Caching in LLM systems?

Caching stores previous responses or computations to reduce repeated processing and improve speed.

---

## ❓ 13. What is Hallucination and how to reduce it?

Hallucination is when the model generates incorrect information.

It can be reduced using RAG, better prompts, and grounding with real data.

---

## ❓ 14. What is Chunking in RAG?

Chunking is breaking large documents into smaller parts before creating embeddings.

This improves retrieval accuracy.

---

## ❓ 15. What is Prompt Chaining?

Prompt chaining is breaking a complex task into multiple smaller prompts and processing them step by step.
